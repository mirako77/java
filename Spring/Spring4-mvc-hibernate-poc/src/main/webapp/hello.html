Welcome Spring MVC!

Spring4-mvc hibernate bootstrap POC Maven Webapp

Vous étes a la page d'accueil hello.html

Ceci est un projet de départ pour créér rapidement
votre nouveau projet => Maven, Spring MVC, hibernate, bootstrap.
Le build se fait dans maven avec: mvn clean package
On doit le lancer dans un serveur d'application, par exemple tomcat.

L'url d'accés : http://localhost:8080/SpringMVCPoc/
Déclencher une action Spring MVC classique: SpringMVCPoc/info/app/hello
Déclencher une action Spring MVC paramétré: SpringMVCPoc/info/app/hello/mick

info locale FR: SpringMVCPoc/app/info/locale?locale=fr
info locale EN: SpringMVCPoc/app/info/locale?locale=en
